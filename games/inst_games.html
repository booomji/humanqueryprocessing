Group 1

Remember: you are pretending to be a robot! Follow the instructions closely, even if they seem unnecessary.
  1. Start your timer
  2. Go row by row and cross out rows if the Number Sold is ≤ 20,000,000
  3. Go row by row; if the row is not crossed out, write down the decade of the Release Date in the "Decades" scratch sheet if it is not already written down
  4. For each decade you wrote down:
    1. Create a counter starting at 0 on the "Counters" scratch sheet
    2. Go row by row; if the row is not crossed out and the Release Date is in the same decade, then add 1 to the decade's counter
  5. Read through the "Counters" scratch sheet and write the decade with the biggest counter in the "Result" table below.
  6. Write down how long it took and let Eugene know by raising your hand

Remember: you are pretending to be a robot! Follow the instructions closely, even if they seem unnecessary.

  2. Each member should do the following
    1. Go row by row and cross out rows if the Number Sold is ≤ 20,000,000
    2. Go row by row; if the row is not crossed out, then:
