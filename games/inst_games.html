Group 1

Remember: you are pretending to be a robot! Follow the instructions closely, even if they seem unnecessary.
  1. Start your timer
  2. Look at each row one by one; cross out the row if the Number Sold is ≤ 20,000,000
  3. Look at each row one by one again; if the row is not crossed out, write the decade of the Release Date in the "Decades" scratch sheet if it is not already written down
  4. Now you have a list of all the unique decades in the "Decades" scratch sheet!
  5. For each decade you wrote down:
    1. Write the decade and the number 0 as a row in the "Counters" scratch sheet
    2. Look at each row in the list of video games; if the row is not crossed out and the Release Date is in the same decade, then add 1 to the decade's number
  6. Read through the "Counters" scratch sheet and write the decade with the biggest counter in the "Result" table below.
  7. Write down how long it took and let Eugene know by raising your hand
ResultHow many minutes did your group take?

Group 3

Remember: you are pretending to be a robot! Follow the instructions closely, even if they seem unnecessary.
  1. Give each member in your 5-person group one of the Data sheets, a copy of the "Counters" scratch sheet, and a copy of these instructions.
  2. Each member should do the following
    1. Look at each row one by one; cross out the row if the Number Sold is ≤ 20,000,000
    2. Look at each row one by one again; if the row is not crossed out, then:
      1. If the decade of the row's Release Date isn't on the "Counters" scratch sheet, write it down in a new row, and write 0 for its counter value.
      2. Add 1 to the decade's counter
  3. Once everyone in your group is done, go through each member's "Counters" scratch sheet and
    1. copy each row to the "Master Counters" scratch sheet.
    2. If the decade already exists, add their counter value together and update the row in the "Moster Counters" scratch sheet
  4. Read through the "Master Counters" scratch sheet and write the decade with the biggest counter in the "Result" table below.
  5. Write down how long it took and let Eugene know by raising your hand
ResultHow many minutes did your group take?
