Single thread, row oriented

Remember: you are pretending to be a robot! Follow the instructions closely, even if they seem unnecessary.
  1. Start your timer
  2. Go row by row and cross out rows if the Gender is not female
  3. Go row by row; if the row is not crossed out, write down birthyear's decade in the "Decades" scratch paper if it is not already written down
  4. For each decade you wrote down:
    1. Create a counter starting at 0 on the "Counters" scratch paper
    2. Go row by row; if the row is not crossed out and the birthyear is in the same decade, then add 1 to the decade's counter
  5. Read through the "Counters" scratch paper and write the decade with the biggest counter in the "Result" table below.
  6. Stop your timer and let Eugene know
